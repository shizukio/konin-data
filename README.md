# KONIN Data

KONIN Data は、高等学校卒業程度認定試験（高卒認定）の学習アプリ **KONIN** で利用するデータを管理するリポジトリです。

このリポジトリでは、過去問題・解答・メタデータなど、アプリが利用する各種データを管理しています。

> このリポジトリはデータ管理を目的としており、アプリケーション本体は含まれていません。

---

## ディレクトリ構成

```text
exams/
└── 2025/
    ├── 1/
    │   ├── kokugo/
    │   │   ├── question.pdf
    │   │   └── answer.pdf
    │   ├── eigo/
    │   └── ...
    └── 2/
```

### ディレクトリ

| 名前       | 説明    |
| -------- | ----- |
| `2025`   | 試験年度  |
| `1`      | 第1回試験 |
| `kokugo` | 科目ID  |

### ファイル

| ファイル           | 説明    |
| -------------- | ----- |
| `question.pdf` | 問題冊子  |
| `answer.pdf`   | 解答・解説 |

---

## スクリプト

### PDFを追加

対話形式で PDF を追加します。

```bash
pnpm add:pdf
```

---

### カタログ生成

`catalog.json` を生成します。

```bash
pnpm generate:catalog
```

---

### メタデータ生成

`metadata.json` を生成します。

```bash
pnpm generate:metadata
```

---

### すべて生成

```bash
pnpm generate
```

---

## catalog.json

`catalog.json` はアプリが試験・科目・PDFの構成を認識するためのデータです。

このファイルは手動で編集せず、スクリプトによって自動生成してください。

---

## metadata.json

`metadata.json` は更新確認や整合性確認のためのデータです。

各ファイルの SHA-256 ハッシュやファイルサイズなどの情報を保持します。

---

## データについて

このリポジトリに含まれる PDF は、文部科学省が公開している高等学校卒業程度認定試験の資料を利用しています。

KONIN は、学習を支援することを目的とした非公式プロジェクトです。
